Welcome to unlint.github.com

This automatic linter is the easiest way to validate code style in any pull request on github.

We support for now:

  • C++ via cpplint.py
  • Java via checkstyle
  • Scala via scalastyle
  • JavaScript via jslint
  • Coffeescript via coffeelint

Usage

Simply paste url of pull request in form below and press Advice button to see you results.
