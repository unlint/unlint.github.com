Welcome to unlint.github.com

This automatic linter is the easiest way to validate code style in any pull request on github. Simply paste url of pull request in form below and press Advice button to see you results.

With authentication (unsafe)

Languages

  • C++ via cpplint.py
  • Java via checkstyle
  • Scala via scalastyle
  • JavaScript via jslint
  • Coffeescript via coffeelint
  • Perl via Perl::Critic
  • CSS via CSS LINT
  • PHP via PHPLint
  • Python via pep8

Browser plugins

  • chrome extension
  • firefox plugin

Support

Not found your language in list? You know how to validate code better? Please fill issue on our github page or even make pull request. Code linter should work fine on linux ecosystem to be integrated in this project.
